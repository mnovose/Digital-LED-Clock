# Digital-LED-Clock (WIP)
Digital clock display using short LED strips, programmed with C++ on an ESP32.
Current progress includes successfully controlling the LED strips, first verision of the code completed (but untested), counting from 0 - 9 tested, and assembly of a prototype. 

  **Hardware**  
 - ESP32
 - 74HC595 shift register
 -  LED strips  

 **Software**  
 - Arduino IDE  
 - Autodesk Fusion 360 
 - Autodesk Tinkercad
